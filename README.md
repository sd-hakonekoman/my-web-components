# Study Components

コンポーネントベースの学習プロジェクトです。再利用可能なUIコンポーネントの研究と実装を通じて、モダンなWeb開発の技術とパターンを学習することを目的としています。

## プロジェクト概要

このプロジェクトは、以下の学習目標を持っています：

- 再利用可能なUIコンポーネントの設計と実装
- コンポーネント合成パターンの理解
- モダンなWeb技術（HTML5、CSS3、ES6+）の活用
- アクセシビリティとユーザビリティの考慮
- テスト駆動開発の実践

## 技術スタック

- **開発環境**: Vite
- **言語**: HTML, CSS, JavaScript (ES6+)
- **スタイリング**: CSS Modules / Vanilla CSS
- **テスト**: 未定（プロジェクト進行に応じて選定）

## ディレクトリ構造

```
study-components/
├── components/          # 再利用可能なUIコンポーネント
```

## セットアップ

### 前提条件

- Node.js (v16以上推奨)
- npm または yarn

### インストール

```bash
# 依存関係をインストール
npm install

# 開発サーバーを起動
npm run dev

# プロダクションビルド
npm run build

# プレビュー
npm run preview
```

## 開発ガイドライン

### コンポーネント作成手順

1. `/components/[ComponentName]/` フォルダを作成
2. メインコンポーネントファイルを実装
3. スタイルファイルを作成
4. 使用例を `/examples/` に追加
5. 必要に応じてテストを作成

### 命名規則

- **コンポーネント**: PascalCase (例: `Button`, `InputField`)
- **ファイル**: コンポーネント名に合わせる
- **CSS クラス**: BEM記法またはkebab-case
- **JavaScript変数**: camelCase
